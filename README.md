# SUDOKU
Project for AP CSA
Stephanie Zhu Period 5

### Description + Summary of how board is generated 
  This project generates a 9x9 sudoku board that is different (probably) everytime. The program starts with an empty board, generates a random number (1-9) for the individual unit in the 2d array, and checks if the unit is valid. To be "valid" means that the number cant appear in the same column, row, and 3x3 so there are checks for that. As this generates, some units will not have a solution so the program undos by setting the prev value to 0 and trying again. 

### How to run
  Clone the repo and run via the run/debug button
